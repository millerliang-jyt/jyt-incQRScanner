# jyt-incQRScanner

## 專案簡介

這是一個基於 Vue.js 的「QR Code／條碼（Barcode）掃描 Web 應用程式」，主要功能如下：

- 讓使用者透過手機或電腦相機掃描 QR Code，進行財產盤點或資產管理。
- 支援盤點人員名單管理，可自訂或新增人員，並記錄每筆掃描的盤點人員與時間。
- 掃描結果會顯示狀態（成功、重複、失敗），並有本地記錄功能，最多保留 300 筆。
- 可選擇設定 Webhook URL，將掃描結果即時傳送到遠端伺服器（如 n8n 流程）。
- 若未設定 Webhook，則僅儲存於本地瀏覽器。
- 介面現代化，採用 Tailwind CSS，並針對行動裝置優化。
- 具備手動輸入資產編號、清除記錄、相機權限與錯誤提示等功能。

## 支援條碼（Barcode）掃描

本系統除 QR Code 外，也支援常見一維條碼（Barcode）掃描，適用於多數資產標籤、商品條碼等。

## 應用場景舉例

- 公司、工廠、學校等資產盤點（財產標籤掃描）
- 倉庫進出貨、物料管理
- 圖書館書籍管理
- 活動報到、票券驗證
- 商品條碼掃描記錄
- 其他需快速記錄編號、身份、物品的場域
